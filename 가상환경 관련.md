# 가상환경 구축 및 앱 실행 가이드

## 1. 가상환경 생성 (Python 3.12)
프로젝트 루트 폴더(예: `MonitoringApp`)에서 아래 명령어를 실행하여 `venv`라는 이름의 가상환경을 생성합니다. 
(Python 3.12 버전이 시스템에 설치되어 있어야 합니다.)

```bash
# Python 3.12 환경을 명시적으로 사용할 경우
python3.12 -m venv venv
```

## 2. 가상환경 실행 (활성화)
**명령 프롬프트(CMD):**
```cmd
venv\Scripts\activate.bat
```
**PowerShell:**
```powershell
venv\Scripts\Activate.ps1
```
> **참고:** 정상적으로 활성화되면 터미널 좌측에 `(venv)`라는 접두사가 표시됩니다.

---

## 3. 패키지 설치 및 앱 실행

가상환경이 활성화된 상태에서, 필요한 패키지를 설치한 후 애플리케이션을 실행합니다.

### 3.1. pip 최신 버전 업데이트
먼저 패키지 관리자인 `pip`를 최신 버전으로 업데이트합니다.
```bash
python -m pip install --upgrade pip
```

### 3.2. 패키지 설치

**방법 A: 요구사항 파일로 자동 설치 (권장)**
```bash
pip install -r requirements.txt
```

**방법 B: 수동 설치**
특정 버전의 `PySide6`와 `pyside6-fluent-widget`을 수동으로 설치해야 한다면, 아래 명령어를 차례대로 입력하세요.
```bash
pip install PySide6==6.10.2
pip install pyside6-fluent-widgets==1.11.1
```

### 3.3. 앱 실행
패키지가 모두 설치되었다면, 아래 명령어로 본 애플리케이션을 실행합니다.
```bash
python main.py
```
